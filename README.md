# Prompt-Powered Kickstart: Beginner Python Toolkit

This project demonstrates how AI prompts can help beginners learn Python and build simple applications.

## Project
AI-Assisted To-Do List Manager

## Features
- Add tasks
- View tasks
- Remove tasks
- Simple command-line interface

## Technologies
- Python
- AI-assisted prompt learning

## How to Run

Run the program using:

python todo_app.py
